My Tech Stack

(So far)

HTML5

The latest evolution of the standard that defines HTML, with new elements, attributes, and behaviors, and a larger set of technologies. It allows the building of more BA Web sites and applications.

CSS3

In a world where the number of people using mobile devices is ever growing, using the latest in CSS allows us to style websites with mobile-first in mind. Not mention make some pretty cool looking pages.

JavaScript

Known as being the programming language of the web, sites just wouldn't have that interactive magic if it wasn't for JavaScript. I use jQuery to write JS more efficiently and have started digging into Angular to create powerful web apps.
